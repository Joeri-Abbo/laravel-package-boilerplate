## Getting Started

### Prerequisites

- Php 8 and up
- Laravel 9

### Installation

To get started clone this project or fork it and add the package to your laravel project.

### Features
-  Blade views inside the package
-  Routing
-  Migrations
-  MVC
-  Tailwind
-  Webpack 
-  pnpm
